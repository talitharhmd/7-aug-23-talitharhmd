LUAS SEGITIGA

Rumus Luas Segitiga Yaitu :

L = 1/2 x a x t

Dimana:
L = Luas
a = Panjang Alas
t = Tinggi


Hitung Luas Segitiga

L = 1/2 x a x t

KELILING SEGITIGA

Rumus Luas Segitiga Yaitu :

K = S1 x S2 x S3

Dimana:
K = Keliling
S1 = SisiA
S2 = SisiB
S3 = SisiC


Hitung Keliling Segitiga

K = S1 + S2 + S3

LUAS JAJARGENJANG

Rumus Luas Jajargenjang Yaitu :

L = a (Alas) x t (Tinggi)

Dimana:
L = Luas
a = Alas
t = Tinggi


Hitung Luas Jajargenjang

L = alas (a) x tinggi (t)

KELILING JAJARGENJANG

Rumus Keliling Jajargenjang Yaitu :

K = 2 x (a + b)

Dimana:
K = Keliling
a = Alas
b = Sisi Miring


Hitung Keliling Jajargenjang

K = 2 x (a + b)
